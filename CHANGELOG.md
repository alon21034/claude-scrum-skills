# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-03-28

- Initial public release of the `sprint` skill package.
- Added Conductor workspace bootstrap helpers: `sprint-setup`, `sprint-board`, and `sprint-finish`.
- Added slash command docs for `/sprint`, `/sprint-task`, `/sprint-board`, and `/sprint-finish`.
- Added host-aware installer support for Codex and Claude paths.
